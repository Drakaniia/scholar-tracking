# Scholarship Amounts by Grade Level and Type

## Grade School (Grades 1-6)

- **Internal Scholarships**: PHP 8,000 - PHP 15,000 per year
- **External Scholarships**: PHP 10,000 - PHP 20,000 per year
- **Examples**:
  - School Grant: PHP 8,000 - PHP 12,000
  - PAEB: PHP 15,000 - PHP 25,000
  - LGU: PHP 10,000 - PHP 18,000

## Junior High School (Grades 7-10)

- **Internal Scholarships**: PHP 10,000 - PHP 20,000 per year
- **External Scholarships**: PHP 12,000 - PHP 25,000 per year
- **Examples**:
  - School Grant: PHP 10,000 - PHP 15,000
  - PAEB: PHP 18,000 - PHP 30,000
  - ESC: PHP 15,000 - PHP 25,000
  - LGU: PHP 12,000 - PHP 22,000

## Senior High School (Grades 11-12)

- **Internal Scholarships**: PHP 12,000 - PHP 25,000 per year
- **External Scholarships**: PHP 15,000 - PHP 30,000 per year
- **Examples**:
  - School Grant: PHP 12,000 - PHP 18,000
  - PAEB: PHP 20,000 - PHP 35,000
  - OLSSEF: PHP 18,000 - PHP 28,000
  - LGU: PHP 15,000 - PHP 25,000

## College/Higher Education

- **Internal Scholarships**: PHP 15,000 - PHP 50,000 per year
- **External Scholarships**: PHP 20,000 - PHP 80,000 per year
- **Examples**:
  - School Grant: PHP 20,000 - PHP 35,000
  - CHED Programs (TES, StuFAPs): PHP 40,000 - PHP 80,000
  - CMSP: PHP 35,000 - PHP 60,000
  - LGU: PHP 25,000 - PHP 45,000
  - CHED-CSP: PHP 50,000 - PHP 100,000

## Scholarship Duration Recommendations

- **Grade School**: 1-6 years (per grade level or block)
- **Junior High**: 4 years (complete JHS)
- **Senior High**: 2 years (complete SHS)
- **College**: 4-5 years (complete degree program)
- **Renewable scholarships**: 1 academic year with renewal based on academic performance

## Common Renewal Criteria

- Maintain minimum GWA (typically 2.0 or 85%)
- Active enrollment status
- Good moral character
- Financial need verification (for need-based scholarships)
